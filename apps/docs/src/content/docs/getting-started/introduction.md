# Minions Component-evals

Pipeline-level benchmarks for RAG, routing, and classification

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-component-evals/sdk`) — Core library for Node.js
- **Python SDK** (`minions-component-evals`) — Python bindings
- **CLI Tool** (`@minions-component-evals/cli`) — Command-line interface
- **Web App** — Interactive playground at [component-evals.minions.wtf](https://component-evals.minions.wtf)
