---
title: Installation
description: Install Minions Component-evals packages
---

## TypeScript / Node.js

```bash
npm install @minions-component-evals/sdk minions-sdk
```

## Python

```bash
pip install minions-component-evals
```

## CLI

```bash
npm install -g @minions-component-evals/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
