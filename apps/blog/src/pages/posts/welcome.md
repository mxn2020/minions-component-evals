# Welcome to Minions Component-evals

We're excited to introduce **Minions Component-evals** — Pipeline-level benchmarks for RAG, routing, and classification.

## What's Included

- **TypeScript SDK**: `npm install @minions-component-evals/sdk`
- **Python SDK**: `pip install minions-component-evals`
- **CLI**: `npm install -g @minions-component-evals/cli`
